# 2주차 -  구현

### 구현 - implementation
-> 머릿속에 있는 알고리즘을 소스코드로 바꾸는 과정
<br>
-> 풀이를 떠올리는 것은 쉽지만 소스코드로 옮기기 어려운 문제


### e.g.

- 알고리즘은 간단, 코드가 길어지는 문제
- 실수 연산, 특정 소수점 자리 출력 문제
- 문자열을 특정 기준에 따라 끊어야 하는 문제
- 적절한 라이브러리 찾아서 풀어야 하는 문제

### 유형 : 서로 유사한 점이 많은 구현 유형들
- 시뮬레이션
- 구현
- 완전탐색
