# 4주차 - 재귀

### 재귀 - Recursive Function

- 자기 자신을 다시 호출하는 함수
<br>

### 특징

- 재귀 함수의 종료 조건을 반드시 명시
- 모든 재귀 함수는 반복문을 이용하여 동일한 기능 구현 가능
- 반복문보다 유리한 경우도 있고 불리한 경우도 있다
- 컴퓨터 메모리의 스택 프레임 이용
